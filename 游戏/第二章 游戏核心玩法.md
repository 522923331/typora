# **第二章 游戏核心玩法（完整版）**

------

# **2.1 职业系统设计（核心玩法的基石）**

本项目采用经典传奇三职业体系，适合单人开发、易于平衡、玩家认知度高。

------

## **2.1.1 战士（Warrior）**

**定位：**近战、单体爆发、高生存
 **特点：**

- 强力近战输出
- 高防御、高生命
- 技能少但关键
- PK 强势

**基础技能：**

1. **攻杀剑法**：近战单体高伤害
2. **烈火剑法**：蓄力攻击，造成巨大爆发
3. **逐日剑法（后期）**：向目标冲刺并造成范围伤害
4. **半月弯刀**：扇形 AOE

**职业特性：**

- 平A 伤害高
- 道具消耗小
- 强化收益明显
- 对装备依赖极高 → 刷装备动机强

------

## **2.1.2 法师（Mage）**

**定位：**远程大范围伤害、清怪速度最快
 **特点：**

- 高输出、低防御
- 范围技能多
- 升级最快，但 PK 脆弱

**基础技能：**

1. **火球术**：基础远程攻击
2. **雷电术**：单体高伤害
3. **冰咆哮**：大范围 AOE
4. **火墙**：持续伤害区域

**职业特性：**

- 刷怪效率最高
- 装备需求相对较低（但极品装备收益巨大）
- 技能组合丰富

------

## **2.1.3 道士（Taoist）**

**定位：召唤、辅助、持续伤害、最全面**
 **特点：**

- 可召唤神兽
- 有治疗与毒
- PK 稳定且灵活

**基础技能：**

1. **施毒术（绿毒/红毒）**
2. **治愈术**
3. **召唤神兽**
4. **灵魂火符**：中距离打击

**职业特性：**

- 兼顾伤害、生存、辅助
- 适合单人游戏体验
- 召唤兽让低装备也能打高怪

------

# **2.2 战斗机制**

------

## **2.2.1 操作方式**

- 虚拟摇杆 + 技能按键
- 自动战斗（可开启/关闭）
- 手动模式更强（战士连击、法师走位躲 AOE）

------

## **2.2.2 战斗表现**

- 挥砍、击退、飘血、暴击
- 爆装备时有震动 + 闪光
- Boss 技能范围显示

------

## **2.2.3 战斗公式（简化版本）**

```
伤害 = (攻击 - 防御/2) × 技能倍率 × 随机波动(0.95–1.05)
```

暴击：

```
暴击伤害 = 普通伤害 × (1.5 + 暴击伤害%)
```

技能伤害举例（烈火剑法）：

```
烈火伤害 = 普通攻击 × 2.2
```

------

# **2.3 PK 系统（传奇核心特色）**

------

## **2.3.1 PK 模式**

- 和平（只攻击怪）
- 组队（不攻击队友）
- 全体（可攻击所有玩家）

------

## **2.3.2 红名机制**

- 击杀玩家 → 获得 1–3 点 PK 值
- PK 值 > 10 → 红名
- 红名：
  - 掉落概率 +50%
  - 城镇 NPC 不理会
  - 红名可在野外被追杀

------

## **2.3.3 掉落规则（简化版）**

被击杀时掉落：

- 1 件非绑定装备（优先掉落身上最低品级）
- 一部分金币
- 红名掉落多 1 件装备

------

# **2.4 掉落爆装机制（最核心部分）**

这是你最喜欢的部分，本游戏会重点强化“爆装备爽点”。

------

## **2.4.1 掉落逻辑**

怪物击杀 → 后端计算掉落池 → 客户端播放掉落动画

掉落池包含：

- 金币
- 材料
- 装备（高概率）
- 稀有装备（极低概率）

------

## **2.4.2 掉落品质概率（默认）**

```
白装：60%
绿装：25%
蓝装：10%
紫装：4%
金装：1%
红装（特效装备）：0.1%
```

（Boss 会有更高概率）

------

## **2.4.3 装备词条随机机制（类似暗黑 + 传奇）**

每件装备有 1–6 条随机词条：

| 品质       | 词条数量   |
| ---------- | ---------- |
| 白         | 0–1        |
| 绿         | 1–2        |
| 蓝         | 2–3        |
| 紫         | 3–4        |
| 金         | 4–5        |
| 红（特效） | 5–6 + 特效 |

------

## **2.4.4 词条类型（将在数据表给你 300+ 条）**

词条类型包含：

### 基础属性词条

- 攻击
- 防御
- 生命

### 战斗词条

- 暴击、暴击伤害
- 穿透、忽略防御
- 攻速、施法速度
- 生命恢复、吸血

### 技能词条

- “烈火剑法 +10% 伤害”
- “冰咆哮范围扩大 20%”

### 稀有词条（红装）

- “攻击时 10% 释放火球（300% 伤害）”
- “移动时每秒恢复 3% 生命”
- “击杀怪物 +5% 再次掉落判定”

这些将在你要的 JSON 文件中全部给出。

# **2.5 强化 / 熔炼系统**

------

## **2.5.1 强化（+1 ~ +15）**

每次强化需要材料 / 金币
 强化加成：攻击、防御、生命等

强化成功率递减（可调）：

| 等级    | 成功率 |
| ------- | ------ |
| +1~+5   | 100%   |
| +6~+9   | 80%    |
| +10~+12 | 50%    |
| +13~+15 | 20–30% |

失败下降 1 级（可选）

------

## **2.5.2 熔炼系统**

熔炼白/绿/蓝装备可获得“熔炼值”，用于升级熔炉等级，提高：

- +额外攻击
- +掉落率
- +自动拾取范围

适合单人长期玩。

------

# **2.6 怪物与 AI**

------

## **2.6.1 普通怪**

特点：

- 血量低
- 自动追踪玩家
- 攻击节奏慢
- 掉落白/绿装备、材料、金币

------

## **2.6.2 精英怪**

特点：

- 边框更亮
- 2~3 条随机词条（强化怪物玩法）
- 高爆率

怪物词条示例：

- 冰冻免疫
- 反弹伤害
- 移速+30%

------

## **2.6.3 Boss**

特点：

- 多阶段
- 技能范围显示
- 保底掉落至少 1 件蓝装

Boss 技能示例：

- 跳斩（战士 Boss）
- 火焰喷吐（龙类 Boss）
- 暴风雪（冰魔 Boss）

------

# **2.7 副本系统（简化版）**

------

## **2.7.1 经验副本**

每天 1 次
 怪物密度大
 通关奖励大量经验

------

## **2.7.2 装备副本**

掉落高品质装备
 可重复刷

------

## **2.7.3 Boss 副本**

Boss 单独出现
 掉落概率高
 挑战次数有限

------

# **2.8 自动战斗（可配置）**

自动战斗包含：

- 自动攻击
- 自动施法
- 自动拾取

开发难度低，适合个人项目。

------

# **2.9 成长系统（轻量化）**

------

## **等级成长**

1–50 级（可扩展至 100）
 经验曲线平滑，保证打怪不累赘

------

## **战力系统**

显示所有属性的综合评分
 用于满足玩家心理满足感

------

## **技能成长**

技能等级 1–10
 技能随等级提升伤害和效果

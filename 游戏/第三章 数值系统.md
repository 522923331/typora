# **3.2 怪物属性模型**

考虑到你是一个人开发，怪物设计应极其统一与可扩展。

------

## **3.2.1 普通怪属性计算（自动生成）**

公式模板：

```
怪物生命 = (玩家平均生命 × 0.5) × 地图系数
怪物攻击 = (玩家平均防御 × 0.7) × 地图系数
怪物经验 = (怪物生命 / 3) × 地图系数
```

地图系数按难度：

- 新手村：0.7
- 普通野外：1.0
- 精英区：1.3
- Boss 区：1.8

------

## **3.2.2 精英怪**

- 血量 × 2
- 攻击 × 1.5
- 掉落率 × 2
- 随机怪物词条 1–3 条

------

## **3.2.3 Boss 属性**

Boss 生命周期曲线目标：

- 适合 20–60 秒战斗
- 输出方式多样

属性公式：

```
BossHP = 玩家平均生命 × 16 × 地图系数
BossAtk = 玩家平均防御 × 2.5 × 地图系数
BossExp = 普通怪 10–15 倍
```

------

# **3.3 升级经验公式**

希望 1–30 级比较快，30–50 级逐渐有压力，但不会卡玩家。

### 推荐经验曲线：

```
Exp(n) = 30 × n^2 + 100 × n
```

表现：

- 1–10 级：极快（5–10 分钟）
- 10–20 级：需要一定怪量
- 20–40 级：稳定成长
- 40–50 级：略微拉长时间（适合深度玩家）

# **3.4 战斗公式（简化、可直接复制用）**

------

## **3.4.1 普通攻击伤害**

```
Damage = (Atk - Def/2) × Random(0.95~1.05)
```

若 Atk < Def/2，至少造成 1 点伤害。

------

## **3.4.2 技能伤害**

```
SkillDamage = Damage × 技能倍率
```

示例：

- 烈火剑法：×2.2
- 冰咆哮：×1.6 AOE

------

## **3.4.3 暴击公式**

```
if random < CritRate:
    Damage = Damage × (1.5 + CritDamage%)
```

------

## **3.4.4 吸血公式**

```
HealHP = Damage × 吸血率
```

------

## **3.4.5 攻速（每秒攻击次数）**

```
APS = 1 + 攻速% 
```

基础攻速 ≈ 1 次 / 秒。

## **3.5.1 装备基础系数（非常重要）**

### （1）武器

攻击主要来自武器：

```
攻击 = 装备等级 × 2.5
```

示例：
 20 级武器 → 20 × 2.5 = 50 攻击（±10%）

------

### （2）防具（头盔、衣服、裤子）

```
防御 = 装备等级 × 1.2
```

------

### （3）项链、戒指

主提供：

- 暴击
- 攻速
- 穿透
- 技能加成

------

### （4）极品词条（将在 JSON 中体现）

词条强度 ≤ 装备基础数值 × 20%
 避免数值溢出。

## **3.7 掉落与爆率体系**

漂亮简单，非常适合你个人开发。

------

## **3.7.1 掉落判定流程**

一次怪物死亡触发三层判定：

### ① 是否掉落装备？

```
dropEquip = Random < equipDropRate
```

equipDropRate（普通怪）：40%
 精英：80%
 Boss：100%

------

### ② 掉落哪种品质？（核心）

```
白：60%
绿：25%
蓝：10%
紫：4%
金：1%
红：0.1%
```

------

### ③ 该品质对应词条数量？

```
白：0–1  
绿：1–2  
蓝：2–3  
紫：3–4  
金：4–5  
红：5–6 + 特效
```

------

## **3.7.2 特效装备掉落**

触发：

```
if Random < 0.001：
    掉落红装特效
```

红装特效将在第 4 章给你完整列表。

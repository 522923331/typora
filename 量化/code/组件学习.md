# @dataclass

`@dataclass` 是 Python 的“**轻量级数据对象生成器**”，类似java中的lombok，自动帮你生成：

- `__init__`
- `__repr__`
- `__eq__`

👉 非常适合：

- 配置对象
- 参数对象
- DTO（数据传输对象）

```python
@dataclass
class BacktestConfig:
    """回测配置
    包含回测所需的所有参数
    """
     # 基本配置
    symbol: str                        # 股票代码
    start_date: str                    # 开始日期 YYYYMMDD
    end_date: str                      # 结束日期 YYYYMMDD
    initial_cash: float = 1000000.0   # 初始资金

```

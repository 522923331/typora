# 模型

## CAPM Model

不是预测神器，而是基准框架。用来衡量超额收益α 。

CAPM（资本资产定价模型）是用于确定资产预期回报率的金融模型，它描述了资产的**预期收益**与**系统性风险**（由 Beta 系数衡量）之间的线性关系：E(R_i) = R_f + β_i × [E(R_m) - R_f]，

也即：股票i的预期收益率 = 无风险收益率 + β系数 × 市场风险溢价。

### R_f 无风险收益率

：完全不承担风险能获得的收益。

```
# 不同市场常用R_f
中国市场：10年期国债收益率 ≈ 2.5-3.0%
美国市场：10年期美债收益率 ≈ 4.0-4.5%
欧洲市场：德国10年期国债 ≈ 2.0-2.5%
```

### 市场风险溢价 E(R_m) - R_f

：投资整个市场（而不是无风险资产）获得的超额回报。

```
# 估计市场风险溢价的三种方法

# 方法1：历史平均法（最常用但有问题）
美国股市历史平均年化回报 ≈ 10%
美国10年国债历史平均 ≈ 4%
历史风险溢价 = 10% - 4% = 6%

# 方法2：隐含法（从当前价格倒推）
使用股息贴现模型或期权隐含波动率

# 方法3：调查法（询问机构投资者预期）
各大投行每年发布预期

# 中国市场的特殊性
1. A股历史短，波动大
2. 政策影响显著
3. 常用参考：5-8%（高于成熟市场）
```

###  **β系数 (Beta)**

这事CAPM量化的灵魂。β的本质理解：

```
β = 1.5  →  市场涨10%，股票涨15%（波动更大）
β = 0.8  →  市场涨10%，股票涨8%（波动更小）
β = -0.5 →  市场涨10%，股票跌5%（反向运动）
```

### CAPM的量化应用

#### 场景1：计算预期收益（选股基础）

```python
def expected_return_capm(rf, beta, market_return):
    """计算CAPM预期收益"""
    return rf + beta * (market_return - rf)

# 实际例子
rf = 0.04  # 无风险利率4%
market_return = 0.10  # 预期市场收益10%
beta_aapl = 1.2  # 苹果β

expected_aapl = expected_return_capm(rf, beta_aapl, market_return)
print(f"苹果预期收益: {expected_aapl:.2%}")  # 11.2%
```

#### 场景2：阿尔法（α）策略（主动管理的核心）

```python
def calculate_alpha(actual_return, rf, beta, market_return):
    """
    计算投资组合的阿尔法
    α = 实际收益 - CAPM预期收益
    """
    expected = expected_return_capm(rf, beta, market_return)
    return actual_return - expected

# 量化对冲基金策略：
# 1. 寻找正α的股票（实际收益 > CAPM预期）
# 2. 做空负α的股票
# 3. 保持市场中性（β ≈ 0）

class AlphaStrategy:
    def __init__(self, rf=0.04, market_return=0.10):
        self.rf = rf
        self.market_return = market_return
    
    def rank_stocks(self, stock_data):
        """根据α排序股票"""
        rankings = []
        for ticker, data in stock_data.items():
            # 计算α
            alpha = calculate_alpha(
                data['actual_return'],
                self.rf,
                data['beta'],
                self.market_return
            )
            
            # 计算信息比率（衡量α的稳定性）
            if data['alpha_std'] > 0:
                info_ratio = alpha / data['alpha_std']
            else:
                info_ratio = 0
            
            rankings.append({
                'ticker': ticker,
                'alpha': alpha,
                'info_ratio': info_ratio,
                'beta': data['beta']
            })
        
        # 按α排序
        return sorted(rankings, key=lambda x: x['alpha'], reverse=True)
```

#### 场景3：绩效评估（区分运气与能力）

```python
def evaluate_fund_performance(fund_returns, market_returns, rf):
    """
    用CAPM评估基金表现
    
    关键指标：
    1. 夏普比率 = (收益 - rf) / 波动率
    2. 特雷诺比率 = (收益 - rf) / β
    3. 詹森阿尔法 = 实际收益 - CAPM预期收益
    """
    # 计算β
    beta = np.cov(fund_returns, market_returns)[0, 1] / np.var(market_returns)
    
    # 夏普比率
    fund_excess = np.mean(fund_returns) - rf
    fund_volatility = np.std(fund_returns)
    sharpe = fund_excess / fund_volatility
    
    # 特雷诺比率
    treynor = fund_excess / beta
    
    # 詹森阿尔法
    market_excess = np.mean(market_returns) - rf
    expected_return = rf + beta * market_excess
    actual_return = np.mean(fund_returns)
    jensen_alpha = actual_return - expected_return
    
    return {
        'beta': beta,
        'sharpe_ratio': sharpe,
        'treynor_ratio': treynor,
        'jensen_alpha': jensen_alpha,
        'market_exposure': beta * 100  # 市场暴露百分比
    }

# 判断基金经理能力：
# α > 0：有选股能力（击败市场）
# α ≈ 0：只是β收益（跟市场走）
# α < 0：跑输市场
```

#### 场景4：风险预算与配置

```python
class RiskBudgeting:
    def __init__(self, target_beta=1.0):
        self.target_beta = target_beta
    
    def calculate_portfolio_beta(self, weights, betas):
        """计算组合β"""
        return np.sum(weights * betas)
    
    def adjust_for_target_beta(self, current_weights, betas):
        """调整权重以达到目标β"""
        current_beta = self.calculate_portfolio_beta(current_weights, betas)
        
        if abs(current_beta - self.target_beta) < 0.01:
            return current_weights
        
        # 需要调整
        adjustment_factor = self.target_beta / current_beta
        adjusted_weights = current_weights * adjustment_factor
        adjusted_weights = adjusted_weights / adjusted_weights.sum()  # 重新归一化
        
        return adjusted_weights
    
    def risk_contribution(self, weights, betas, cov_matrix):
        """计算各资产的风险贡献"""
        portfolio_vol = np.sqrt(weights.T @ cov_matrix @ weights)
        marginal_risk = cov_matrix @ weights / portfolio_vol
        risk_contributions = weights * marginal_risk
        
        return risk_contributions / portfolio_vol  # 百分比形式
```

# **第一章 游戏概述（完整版）**

------

# **1.1 游戏名称（暂定）**

**《烈焰·传承》**
 （说明：名称可替换成 *复古传奇、烈火战域、热血亡灵、永恒沙城* 等，更换不会影响设计）

------

# **1.2 游戏类型**

- **类型定位**：轻量 MMORPG + ARPG 刷宝（复古传奇类）
- **核心标签**：爆装备、PK、升级、轻社交、战法道、红名、掉落

------

# **1.3 核心玩法总结**

本游戏是一个 **高爆率、高数值爽点、低开发成本** 的复古“传奇类”MMORPG。

核心循环为：

**刷怪 → 爆装备 → 提升战力 → 挑战更强怪物 → PK / 抢 Boss → 收集极品属性装备**

特点：

- 经典三职业（战士、法师、道士）
- 刷怪掉落随机装备、随机词条（暗黑类）
- PK 爽感、红名系统
- 世界 Boss 争夺
- 轻社交，不依赖强多人支撑
- 时长需求低，适合个人开发
- 可支持挂机 / 半自动战斗

------

# **1.4 世界观与故事背景（可直接用于文案）**

### **背景摘要**

千年前，“烈焰大陆”被神魔大战撕裂，神族消失，魔族沉眠。
 大陆上只留下三大传承：武者的战气、法师的元素、道士的灵印。

魔族近期复苏，邪物在各地聚集，正在吞噬生命力。
 玩家将加入战气、元素或灵印传承者的行列，从新手村出发，强化装备、提升力量，对抗魔族力量，并最终争夺沙城的统治权。

### **世界基调**

- 中古奇幻（与传奇一致）
- 弱剧情、强玩法
- 世界破败、地图荒野风、怪物以魔化生物为主

------

# **1.5 玩家核心体验（重点）**

玩家进入游戏后，将获得：

### **1. 爽快的战斗体验**

- 从 5 级开始就能快速击杀大量怪物
- 经典传奇打击感：弹刀、飘血、连击
- 战士贴脸、法师AOE、道士召唤宠物

### **2. 刷装爽点（你的最爱）**

- 装备永远有惊喜
- 稀有装备掉落时全场公告
- 装备属性完全随机组合（类似暗黑）
- 词条有极品与偏差，高端玩家愿意反复刷

### **3. PK 爽感**

- 玩家夺怪、抢 Boss
- 红名处罚简单但刺激
- 区域分级：安全区 → 战斗区 → 深红区

### **4. 数值成长快，反馈强**

- 升级速度快
- 战力提升直观
- 玩家可在短时间感受到变强

### **5. 多数系统轻量化，适合个人开发**

- 小世界 / 小地图结构
- 最小依赖社交即可运转
- 世界 Boss 也能单人挑战（削弱版）

------

# **1.6 游戏目标（基于个人开发者）**

为保证项目在 8–12 周内能完成最小可玩版本（MVP），设定如下目标：

### **目标 1：做一个“能玩”的传奇类游戏**

包含：

- 三职业基础技能
- 基础战斗
- 4–6 张野外地图
- 1–2 个 Boss
- 装备系统（随机词条 + 品质）
- 背包、强化、熔炼
- 全流程可玩（从新手村 → 过图 → 爆装）

### **目标 2：重点突出“装备词条 + 掉落爽感”**

- 随机词条
- 品质（白/绿/蓝/紫/金）
- 特效装备（红色）
- 掉落特效（光柱、音效）
- 最小成本打造最大爽点

### **目标 3：轻量 MMORPG（可单人/可未来扩多人）**

- 初期可单人刷图
- 大部分逻辑可前端执行
- 必要部分走后端（掉落、角色状态）
- 后期可支持更多玩家 / 更多服务器

### **目标 4：架构可扩展**

你的未来版本可以新增：

- 行会
- 攻城战
- 宠物
- 副本
- 元素词条
- 深渊地图

------

# **1.7 项目开发策略（适合你个人）**

### **用最小成本实现“传奇爽感”**

- 不做大地图
- 不做复杂社交
- 不做超大数值面板
- 不做几十个技能
- 不做绑定装备/交易系统（可后期）

优先开发：

### **🥇 核心玩法优先：战斗 + 掉落 + 装备随机**

这是传奇类游戏的命脉。

### **🥈 后端轻量化**

- 所有战斗都在前端模拟
- 后端只保存数据、验证掉落
- 小型 Go/Node/Java 服务即可稳定运行

### **🥉 单人即可测试**

开发周期短，压力小，风险低。

------

# **1.8 MVP（最小可行性版本）内容范围**

预计开发 8–12 周，包含：

- 三职业
- 15 个基础技能
- 50–80 件装备（含稀有词条）
- 300 条词条（系统支持）
- 3 张野外地图 + 2 张精英区
- 1 个世界 Boss
- 背包、强化、熔炼
- 自动战斗（简易）
- 登录/注册/角色系统
- 怪物 AI（简易：巡逻 + 攻击）
- 掉落系统（光柱 + 音效）
